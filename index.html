Básico do HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento:

        <!DOCTYPE html>

O documento HTML em si começa com <html> e termina com </html>.

Por exemplo:

        <html>
  ...
</html>

A parte visível do documento HTML é entre <body> e </body>.

        <body>
  ...
</body>

Elementos HTML

Um elemento HTML é definido por uma tag de início.

O elemento HTML é tudo, desde a tag inicial até a tag final:

Por exemplo:

    <tagname>O conteúdo vai aqui...</tagname>

    <h1>Meu primeiro cabeçalho.</h1>
<p>O meu primeiro parágrafo.</p>

Atributos HTML

Os atributos HTML fornecem informações adicionais sobre elementos HTML.

Os atributos são sempre especificados em a tag de início.

Os atributos geralmente vêm em pares nome/valor como: name="valor".

Por exemplo:

A tag <a> define um hiperlink. O atributo href especifica o URL da página para onde o link direciona:

        <a href="https://www.w3schools.com">Visite W3Schools</a>

HTML Headings

Os cabeçalhos HTML são títulos ou legendas que você deseja exibir em uma página da web.

Os cabeçalhos HTML são definidos com as tags <h1> até <h6>.

<h1> define o título mais importante. <h6> define o título menos importante.

Por exemplo:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

HTML Parágrafos

Um parágrafo sempre começa em uma nova linha e geralmente é um bloco de texto.

O elemento HTML <p> define um parágrafo.
Um parágrafo sempre começa em uma nova linha, e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

Por exemplo:

<p> Este é um parágrafo.</p>

<p>Este é um outro parágrafo..<p>

HTML Styles

O atributo HTML Style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Por exemplo:

O atributo <style> tem a seguinte sintaxe:

        <tagname style="propriedade:valor;">

A propriedade é uma propriedade CSS. O valor é um valor CSS.

        <p style="color:red; font-size:16px;">Este é um parágrafo com estilo.</p>

Formatação HTML

HTML contém vários elementos para definir o texto com um significado especial.

Elementos de formatação foram projetados para exibir tipos especiais de texto:

Por exemplo:

  • <b>- Texto em negrito
  • <strong> - Texto importante
  • <i>- Texto em itálico
  • <em>- Texto enfatizado
  • <mark>- Texto marcado
  • <small>- Texto menor
  • <del>- Texto excluído
  • <ins>- Texto inserido
  • <sub>- Texto subscrito
  • <sup>- Texto sobrescrito

HTML Quotation e Citação

HTML <q> para citações curtas

A tag HTML <q> define uma citação curta. Os navegadores normalmente inserem aspas ao redor da citação.

Por exemplo:

<p>WWF's goal is to: <q> Build a future where people live in harmony with nature.</q></p>

HTML <abbr> para abreviações

Define uma abreviação ou um acrônimo, como "HTML", "CSS", "Sr.", "Dr.".
A marcação de abreviações pode fornecer informações úteis aos navegadores, tradução sistemas e mecanismos de pesquisa.

Por exemplo:

<p> The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

HTML Comentários

Os comentários HTML não são exibidos no navegador, mas podem ajude a documentar seu código-fonte HTML.

Por exemplo:

<!-- Escreva seu comentário aqui -->

Observe que há um ponto de exclamação (!) na tag inicial, mas não na tag inicial etiqueta final.

Comentários podem ser usados para ocultar conteúdo.

HTML Cores

As cores HTML são especificadas com nomes de cores predefinidos ou com valores RGB, HEX, HSL, RGBA ou HSLA.
Em HTML, uma cor pode ser especificada usando o nome dela, bem como pode-se definir a cor do texto, das bordas e a cor de fundo para elementos HTML.

Exemplo de códigos:

  • <h1 style="background-color:DodgerBlue;">Hello World</h1>
  • <h1 style="color:Tomato;">Hello World</h1>
  • <h1 style="border:2px solid Tomato;">Hello World</h1>

  • <h1 style="background-color:rgb(255, 99, 71);">...</h1>
  • <h1 style="background-color:#ff6347;">...</h1>
  • <h1 style="background-color:hsl(9, 100%, 64%);">...</h1>
  • <h1 style="background-color:rgba(255, 99, 71, 0.5);">...</h1>
  • <h1 style="background-color:hsla(9, 100%, 64%, 0.5);">...</h1>

    Os links são encontrados em quase todas as páginas da web. Os links permitem que os usuários cliquem de página para página.

    Links HTML são hiperlinks. Você pode clicar em um link e pular para outro documento.
    Quando você move o mouse sobre um link, a seta do mouse se transforma em uma pequena mão.

    O HTML <a> tag define um hiperlink. Tem a seguinte sintaxe:

    <a href="url">link text</a>

    O atributo mais importante do <a> o elemento é o href atributo, que indica o destino do link.
    O texto do link é a parte que será visível para o leitor. Ao clicar no texto do link, o leitor será enviado para o endereço URL especificado.

    HTML Favicon

    Um favicon é uma pequena imagem exibida ao lado do título da página na guia do navegador.

    Você pode usar qualquer imagem que você gosta como seu favicon. Você também pode criar o seu próprio favicon.
    Por ser uma imagem pequena, deve ser uma imagem simples com alto contraste.

    Adicione um <link> elemento para o seu arquivo "index.html", depois do <title> elemento, assim:

    <title>My Page Title</title>
    <link rel="icon" type="image/x-icon" href="/images/favicon.ico">

    Bloco HTML e elementos inline

    Cada elemento HTML possui um valor de exibição padrão, dependendo do tipo de elemento.

    Os dois valores de exibição mais comuns são bloco e inline.

    Elementos em nível de bloco:

    Um elemento em nível de bloco sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço (uma margem) antes e depois do elemento.

    Um elemento de nível de bloco sempre ocupa toda a largura disponível (estende-se para a esquerda e para a direita tanto quanto possível).

    Dois elementos de bloco comumente usados ​​são: <p> e <div>.

    O <div>elemento define uma divisão ou seção em um documento HTML

    Elementos inline:

    Um elemento inline não começa em uma nova linha. Um elemento inline ocupa apenas a largura necessária.

    Por exemplo:

    <span>Hello World</span>

    HTML Div

    O <div> elemento é usado como contêiner para outros elementos HTML.

    O <div>elemento é por padrão um elemento de bloco, o que significa que ocupa toda a largura disponível e vem com quebras de linha antes e depois.

    O <div>elemento não possui atributos obrigatórios, mas style, class e id são comuns.

    Ele também é frequentemente usado para agrupar seções de uma página da web.

    Por exemplo:

    <div>
    <h2>London</h2>
    <p>London is the capital city of England.</p>
    <p>London has over 13 million inhabitants.</p>
    </div>
    </p>

    Alinhar ao centro um elemento <div>:

    Por exemplo:

    <style>
    div {
    width:300px;
    margin:auto;
    }
    </style>

    Você também pode ter muitos <div> contêineres na mesma página.

    Float:

    A propriedade float no CSS não foi originalmente criada para alinhar <div> elementos lado a lado, mas tem sido usada para esse propósito há muitos anos.

    A propriedade CSS float é usada para posicionar e formatar conteúdo e permite que os elementos flutuem próximos uns dos outros em vez de uns sobre os outros.

    <style>
    .mycontainer {
    width:100%;
    overflow:auto;
    }

    .mycontainer div {
    width:33%;
    float:left;
    }
    </style>

    HTML Class

    O HTML class atributo é usado para especificar uma classe para um elemento HTML. Vários elementos HTML podem compartilhar a mesma classe.

    O class atributo é frequentemente usado para apontar para um nome de classe em uma style sheet. Ele também pode ser usado por JavaScript para acessar e manipular elementos com o nome específico da classe.

    Por exemplo:

    <style>
    .city {
    background-color: tomato;
    color: white;
    border: 2px solid black;
    margin: 20px;
    padding: 20px;
    }

    </style>
    <div class="city">
    <h2>London</h2>

    </div>

    HTML JavaScript

    JavaScript torna as páginas HTML mais dinâmicas e interativas.

    O HTML <script> tag é usado para definir um script do lado do cliente (JavaScript).

    O <script> elemento contém instruções de script ou aponta para um arquivo de script externo através do src atributo.

    Usos comuns para JavaScript são manipulação de imagem, validação de formulário e mudanças dinâmicas de conteúdo.

    Para selecionar um elemento HTML, o JavaScript geralmente usa o document.getElementById() método.

    Exemplo de código:

    <script>
    document.getElementById("demo").innerHTML = "Hello JavaScript!";
    </script>

    HTML Charset

    Esta parte explica o conceito de conjunto de caracteres (charset) em HTML e como ele é usado para definir a codificação de caracteres de um documento HTML.

    O conjunto de caracteres padrão para documentos HTML é UTF-8, que suporta uma ampla gama de caracteres de diferentes idiomas e símbolos.

    Você pode definir o conjunto de caracteres em seu documento HTML usando a tag meta com o atributo charset:

    HTML vs. XHTML

    XHTML é uma versão mais rigorosa e estruturada do HTML, que segue as regras da sintaxe XML.

    O HTML é mais tolerante a erros de sintaxe, o XHTML é menos tolerante e pode resultar em erros se a sintaxe não estiver correta.

    Este é um exemplo de um documento HTML:

    <!DOCTYPE html>
<html>
<head>
    <title>HTML Document</title>
</head>
<body>

    <h1>Título Principal</h1>
    <p>Este é um parágrafo em um documento HTML.</p>

</body>
</html>

    este é um exemplo de um documento XHTML equivalente:

    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>XHTML Document</title>
</head>
<body>

    <h1>Título Principal</h1>
    <p>Este é um parágrafo em um documento XHTML.</p>

</body>
</html>

    HTML Canvas

    O elemento HTML <canvas> é usado para desenhar gráficos, em tempo real, via script (geralmente JavaScript).

    Desenho de um Retângulo

    Um retângulo é desenhado com quatro linhas. Para desenhar um retângulo, use as seguintes propriedades e métodos:

    • `beginPath()`: Inicia um caminho.
    • `moveTo()`: Define o ponto de partida do caminho.
    • `lineTo()`: Define o ponto final do caminho.
    • `closePath()`: Cria um caminho de volta ao ponto de partida.
    • `stroke()`: Desenha as bordas do retângulo.

    Desenho de um Gradiente

    Você pode usar gradientes para preencher formas desenhadas em um <canvas>:

    • `createLinearGradient(x,y,x1,y1)`: Cria um objeto de gradiente linear.
    • `addColorStop()`: Especifica uma cor para parar em um gradiente.

    HTML SVG

    SVG define gráficos baseados em vetores em XML, que pode ser incorporado diretamente em páginas HTML.
    Os gráficos SVG são escaláveis e não perdem qualidade se forem ampliados ou redimensionados:

    Exemplo de Retângulo SVG:

    <svg width="400" height="110">
    <rect width="300" height="100" style="fill:rgb(0,0,255);stroke-width:3;stroke:rgb(0,0,0)" />
    </svg>

    Exemplo de Círculo SVG:

    <svg width="400" height="110">
    <circle cx="200" cy="55" r="50" stroke="black" stroke-width="2" fill="red" />
    </svg>

    HTML Multimídia

    Multimídia na web é som, música, vídeos, filmes e animações.

    Multimídia vem em muitos formatos diferentes. Pode ser quase tudo o que você puder ouvir ou ver, como imagens, música, etc, som, vídeos, discos, filmes, animações e muito mais.
    As páginas da Web geralmente contêm elementos multimídia de diferentes tipos e formatos.

    Vídeo HTML

    <video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    Your browser does not support the video tag.
  </video>

    Audio HTML

    <audio controls>
    <source src="horse.mp3" type="audio/mpeg">
    <source src="horse.ogg" type="audio/ogg">
    Your browser does not support the audio element.
  </audio>
